# Twenty Twenty Child

This is a Child Theme for the Twenty-Twenty-Theme from wordpress.


## Installation

1. Click on the "**Design** > Themes" item in your Dashboard
2. Go to "Add Themes"
3. Click on "Upload Theme"
4. Upload the Theme and enjoy :)
